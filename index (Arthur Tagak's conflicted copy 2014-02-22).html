Help

  1. Pick the category of your first pitstop.
  2. Choose "quickest", "shortest", "cheapest", or "specify"
  3. If "quickest", "shortest", or "cheapest" was chosen, pitstop will be added to queue.
  4. If not finished start from step 1 again. If "specify" was chosen go to step 4. If finished adding queues click ROUTE!
  5. After choosing a category, the app will find locations suiting the category. Pick a location to add to the queue.
  6. Alternatively, enter a location manually in search bar under "other" category.
  7. Once pitstop is fully determined, if you know this is your final pitstop, you may select "last pit stop, route now".
  8. Otherwise, select "add and return to pitstops" to view pit stop categories and stop queue, and optionally edit previously added pitstops. When finished, select "route".
